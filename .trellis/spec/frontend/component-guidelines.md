# Component Guidelines

## Status

There are no frontend components in this project.

## Convention

- Do not introduce React/Vue/Svelte components for course-paper or COMSOL analysis tasks.
- If a future task requests a UI, create a new Trellis task and define frontend conventions at that time.
- Static manuscript figures should be generated by Python/COMSOL scripts and referenced from Markdown.

## Forbidden Patterns

- Do not convert scientific figures into ad hoc UI components.
- Do not add frontend dependencies just to display static plots.
