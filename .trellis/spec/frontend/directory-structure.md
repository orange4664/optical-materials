# Frontend Directory Structure

## Status

No frontend application exists in this workspace. There are no pages, components, hooks, routes, build system, or browser UI files.

## Convention

- Do not create frontend directories such as `src/components`, `app/`, or `pages/` unless a future task explicitly asks for a web UI.
- For paper figures, use `outputs/comsol/` or `outputs/python/`, not a frontend asset directory.
- For manuscript text, use `course_paper.md` or future document-export files.

## Examples

- Current visual artifacts are static PNGs under `outputs/`.
- Current executable work is Python analysis under `scripts/`.
